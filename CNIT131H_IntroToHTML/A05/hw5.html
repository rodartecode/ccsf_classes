Another Table!

HTML
Hypertext Markup Language is the standard mark-up language that web browsers use to interpret webpage layouts and presentation.
CSS
Cascading Style Sheets is the standard adopted by web browsers to define the style of all the different elements present on a webpage.
FTP
File Transfer Protocol is the networking standard used to transfer files between a client and server over a computer network
SSH
Secure Shell is a network protocol for operating networks over unsecure connections. Most typically its used to allow users to log in to systems and use its services remotely
