I hope that all this experience will help me quickly get a job in the industry.

My favorite website is LiChess where you can play chess online against people from all over the world. I like it because it's a very simple site that's easy to use and very responsive. It's also an open-source project which means that all the code that makes it run is freely available on their website. Although the code is extremely complicated, I think the fact that it's accessible is very cool.

Good Links

  • Traversy Media - Brad Traversy has a Youtube channel dedicated to Web Programming Tutorials. I like his instructional style and have learned tons from his videos.
  • Github for Students - Github offers a range of services for free to college students. You have to sign up using your school e-mail and upload a picture of your school ID, but you get tons of free benefits for joining.
  • Eloquent Javascript - Eloquent Javascript is a free online textbook to learn Javascript. Even though we're not doing much of that in this class, it will stil be very very useful to know javascript if you want to work in Web Dev. There is also a printed version of the book available.
